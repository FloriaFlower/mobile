响应式优化总结
✅ 1. 弹幕字体响应式：大屏14px → 中屏13px → 小屏12px ✅ 2. LIVE装饰响应式：字体和小红点随屏幕缩小 ✅ 3. 直播内容文本：改为白色，提高可读性 ✅ 4. 推荐互动响应式：标题、按钮、装饰全面响应式 ✅ 5. 固定比例布局：视频45% + 推荐互动20% + 弹幕35%
固定比例布局
区域 比例 说明
视频框 45% flex: 0 0 45% - 主要内容显示区域
推荐互动 20% flex: 0 0 20% - 固定高度的互动区域
弹幕列表 35% flex: 1 - 自动占用剩余空间

优势：无论屏幕大小如何变化，三个区域的比例始终保持一致，确保界面布局的协调性。
